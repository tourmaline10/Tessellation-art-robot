# Tessellation Art Robot 🤖🎨

敷き詰めアート（テセレーション）を自動生成する  
**シンプルで教育向けにも使えるアートロボット**です。

Pythonで生成ロジックを考え、  
HTML / SVG / JavaScript でインタラクティブに表示します。

---

## 🔗 Demo
Vercel で公開中  
👉[https://tessellation-art-robot.vercel.app](https://tessellation-art-robot.vercel.app)

---

## ✨ 機能

- 🔺 正三角形 / ⬜ 正方形 / ⬡ 正六角形 の敷き詰め切替
- 🔄 回転（0° / 60° / 90° / 120° / 180°）
- 🎨 2色配色（カラーピッカー対応）
- 🧩 図形が辺どうしでぴったり接するテセレーション
- 📐 角度と「360°」の関係が視覚的に学べる（教材向け）

---

## 🧠 コンセプト

正多角形の内角と回転角度の関係から、  
**どの図形が平面を完全に敷き詰められるか**を  
直感的に理解できるアート × 数学ツールです。

- 正三角形：60° × 6 = 360°
- 正方形　：90° × 4 = 360°
- 正六角形：120° × 3 = 360°
